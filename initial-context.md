# Initial Project Context

## Project Overview

FinanzApp is a personal finance management application designed to help users track their income, expenses, and investments. The platform provides powerful analytics, interactive dashboards, and tools for managing recurring transactions. It aims to simplify financial management for individuals and small businesses.

## Tech Stack

### Frontend

- **Next.js 16**: Modern React framework with App Router.
- **React 19**: For building dynamic user interfaces.
- **TypeScript**: Ensures type safety and better developer experience.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Radix UI**: Accessible and customizable UI components.
- **Chart.js / Recharts**: For data visualization.

### Backend

- **Next.js API Routes**: Server-side logic and API endpoints.
- **PostgreSQL**: Relational database for storing financial data.
- **NextAuth.js**: Secure authentication system.

### Development Tools

- **ESLint**: For code quality and consistency.
- **Prettier**: Code formatting.
- **Jest**: Unit testing framework.
- **Playwright**: End-to-end testing.
- **tsx**: TypeScript execution for scripts.

## Project State

- **Current Stage**: Early-stage development.
- **Milestones**:
  - MVP Launch: Q1 2026.
  - Beta Testing: Q2 2026.
  - Full Release: Q3 2026.

## Key Features

- **Transaction Management**: Track income, expenses, and investments.
- **Recurring Transactions**: Automate recurring financial entries.
- **Analytics Dashboard**: Visualize financial data with charts and graphs.
- **Export Capabilities**: Export data to Excel for further analysis.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Secure Authentication**: Protect user data with NextAuth.js.

## Constraints and Priorities

- **Constraints**:

  - Limited team size may impact development speed.
  - Dependency on PostgreSQL for database operations.
  - Ensuring accessibility and performance across devices.

- **Priorities**:
  - Delivering a stable and functional MVP by Q1 2026.
  - Ensuring data security and user privacy.
  - Providing a seamless user experience.
